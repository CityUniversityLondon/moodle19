The imoodel theme

About

The imoodel theme was developed by Delistavrou Constantinos in 2008 as part of his MSc thesis at the University of Macedonia in Thessaloniki, Greece.

It is based on Moodle's default theme but it is using chalk and light green colors.

The imoodel theme is using AJAX technology through the Yahoo! User Interface library.

It provides each system's user (not just the administrator) with the capability of moving Moodle menu panels around the web-top and saving their position so that the next time that he/she will visit the same page the panels will be placed where they were saved to be.

The imoodel funciotnality is not provided to a system's visitor users nor when a user is not logged in.

The imoodel theme was developed using YUI library v2.5.2 until July 2008, so functionality such as window resizing that was yet in beta for the YUI is not suppοrted. The imoodel theme was developed for demonstrating AJAX capabilities in Moodle's environment. As YUI evolves imoodel will follow adding extra functionality.

Installation

Please select the following link to create the data table that is used by the imoodel theme for saving panels' positions in Moodle's database.

Install imoodel data table. A success message will apear on a new tab/window. If there is no message means that table imoodeldata already exists and the imoodel theme is fully functional.

Το θέμα imoodel

Σχετικά

Το θέμα imoodel αναπτύχθηκε από τον Δελησταύρου Κωνσταντίνο το 2008 ως τμήμα της διπλωματικής εργασίας του για το πρόγραμμα μεταπτυχιακών σπουδών του Τμήματος Εφαρμοσμένης Πληροφορικής του Πανεπιστημίου Μακεδονίας στη Θεσσαλονίκη.

Βασίζεται στο θέμα default του Moodle με αλλαγμένα τα προεπιλεγμένα χρώματα προς το ανοιχτό κίτρινο και πράσινο.

Το θέμα imoodel χρησιμοποιεί τεχνολογία AJAX μέσω της βιβλιοθήκης Yahoo! User Interface.

Παρέχει σε κάθε χρήστη ενός συστήματος (και όχι μόνο στον διαχειριστή) τη δυνατότητα να μετακινεί τα πλαίσια των μενού του Moodle στην επιφάνεια εργασίας του φυλλομετρητή και να αποθηκεύει τις θέσεις τους έτσι ώστε την επόμενη φορά που θα επισκεφθεί την ίδια σελίδα, τα πλαίσια θα τοποθετηθούν στην τελευταία θέση που αποθηκεύτηκαν.

Η λειτουργικότητα του θέματος imoodel δεν παρέχεται στους επισκέπτες ενός συστήματος παρά μόνο στους εγγεγραμμένους χρήστες του όταν συνδεθούν σε αυτό.

Το θέμα imoodel αναπτύχθηκε έως τον Ιούλιο του 2008, χρησιμοποιώντας την έκδοση v2.5.2 της βιβλιοθήκης YUI. Για το λόγο αυτό δεν διαθέτει λειτουργίες όπως η αλλαγή μεγέθους των παραθύρων η οποία βρισκόταν σε δοκιμαστικό βήτα στάδιο από τη Yahoo! και δεν υποστηριζόταν στο Moodle. Το θέμα imoodel δημιουργήθηκε για την επίδειξη της εφαρμογής της τεχνολογίας AJAX στο περιβάλλον του Moodle. Καθώς η βιβλιοθήκη YUI θα εξελίσσεται, το θέμα imoodel θα ακολουθεί με προσθήκη νέων λειτουργιών.
